# RESEARCH FINDINGS — StormBreaker Studios

**Agent:** Researcher (Player-Intelligence)
**Input:** 2.04M installs · 19.8k payers · €0.03 ARPU · <1% conversion

---

## 3 Critical Problems

### PROBLEM 1: 69% Never See Day 2
- Median session: **47 seconds**
- D1 retention: **~31%** (69% churn before Day 2)
- Players churn before any monetisation touchpoint
- Core loop lacks early emotional spike

### PROBLEM 2: Monetisation Architecturally Invisible
- Shop unlocks at **Level 5**
- Only **34%** of installs reach Level 5
- **1.3M players** never see the option to spend
- This is a placement problem, not a pricing problem

### PROBLEM 3: Zero Personalisation
- **2M players** treated identically
- Same shop, same price, same push cadence
- No behavioural segmentation
- No dynamic pricing or engagement triggers
