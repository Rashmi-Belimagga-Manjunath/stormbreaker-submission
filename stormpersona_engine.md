## 3 Mechanics · 3 Segments

### A: Flashpoint — Sprinters (66% of players)
- Hero-moment VFX in first **30 seconds**
- Cosmetic reward locked behind **Level 5 gate** with reminder UI
- **€0.99** impulse purchase
- **Target:** 47s → 90s median session
- **KPI:** D0→D1 retention +8pp

### B: Re-engagement Carousel — Floaters (30% of players)
- Personalised push notification at **4-hour gap**
- Deep-link to near-miss reward state
- **€1.99** bundle
- **Target:** Push TTR ≥25%
- **KPI:** D7 retention +8pp

### C: Curated Boutique — Pillars (4% of players)
- Affinity-ranked **6-item personalised shop**
- C1: €0.49–0.99 hook item
- C2: €3.99–7.99 premium bundles
- Flash Sale: 40% off (capped at 25% until margin model validated)
- **KPI:** C1 conversion ≥1.6%

---

## Player Journey

```
Install → Segment (Sprinters/Floaters/Pillars)
  → Mechanic assigned
  → Purchase event → Feedback loop → Re-segment
```
